Local Environments

Our Local Development Environment. Made use of Docker and Docker Composer to create a Linux based VM containing both a stock Wordpress Container for the site, and a Mysql container for the Database. The benefit of this was that in order to set up our entire environment, all we needed to do was distribute a text file that contained the instructions for the Docker Composer.
